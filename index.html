GXBot

GXBot is an advanced Discord bot designed to provide full server management, moderation, automation, economy systems, and interactive features across multiple servers.

Bot Capabilities

General Features
  • Server statistics and information
  • Account verification system
  • Global hub network across servers
  • User profile and personal server data
  • Bot information and ownership details
  • Voting and Top.gg integration
Member Management & Moderation
  • Warning system with history tracking
  • Kick, ban, mute, and timeout controls
  • Role management for individual members
  • Mass role assignment and removal
  • Full moderation control tools
Channel & Room Control
  • Lock and unlock channels
  • Clear messages and manage content
  • Hide or show channels
  • Enable or block links
  • Restrict bot commands in specific rooms
  • Channel and role removal tools
Advanced Configuration & Automation
  • Logging system (messages, deletions, invites)
  • Welcome system setup
  • Suggestions system
  • Ticket system with points tracking
  • Auto-reply and smart responses
  • Custom aliases and command shortcuts
  • Rating system enable/disable
  • Invite tracking and reset
Economy System
  • Virtual bank and user profiles
  • Currency transfers between members
  • Coupons creation and usage
  • Trust rewards system
  • Background customization
  • Leaderboards for users, servers, and tasks
Entertainment & Interaction
  • GX Wheel (luck-based game)
  • Random opinions and judgments
  • Avatar display tools
Additional Utilities
  • Islamic adhkar system
  • Embed message creation
  • Emoji automation in channels
  • Usage statistics tracking
  • Tax room configuration
